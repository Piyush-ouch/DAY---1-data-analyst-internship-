4. Live NAV Integration (API)

To acquire up-to-date Mutual Fund Net Asset Values, live_nav_fetch.py requests live JSON feeds from https://api.mfapi.in.

Execution Flow

  1. Retrieves scheme histories for HDFC Top 100 Direct (125497) and 5 Bluechip/Large Cap schemes (119551, 120503, 118632, 119092, 120841).
  2. Implements a 3-stage retry loop with progressive timeouts (20s, 30s, 40s) and a delay period to safely handle transient network timeouts.
  3. Extracts meta fields and series details, mapping them to structured DataFrames.
  4. Saves individual CSV files as well as a merged master dataset live_nav_fetched_all.csv inside data/raw/.
Critical Data Mappings (Anomalies Found)
The API codes supplied in the assignment show a mismatch with the actual fund names returned by the official API:
  • Code 125497: Labeled as HDFC Top 100 Direct -> API returns SBI Small Cap Fund - Direct Plan - Growth
  • Code 119551: Labeled as SBI Bluechip -> API returns Aditya Birla Sun Life Banking & PSU Debt Fund - DIRECT - IDCW
  • Code 120503: Labeled as ICICI Bluechip -> API returns Axis ELSS Tax Saver Fund - Direct Plan - Growth Option
  • Code 119092: Labeled as Axis Bluechip -> API returns HDFC Money Market Fund - Growth Option - Direct Plan
  • Code 120841: Labeled as Kotak Bluechip -> API returns quant Mid Cap Fund - Growth Option - Direct Plan
