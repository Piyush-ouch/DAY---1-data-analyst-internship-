Robust Data Retainer

  • Target Schemes: SBI, HDFC, ICICI, Nippon, Axis, and Kotak bluechip classes.
  • Network Fail-Safe: Script uses a 3-attempt connection cycle with escalating timeouts (20s -> 30s -> 40s) and delay timers to eliminate read timeouts.
  • Output Structure: Exports both individual CSV records and a single merged dataset `live_nav_fetched_all.csv` inside raw data folders.

Anomalies: Upstream API Mismatches

Official Mappings (mfapi.in)

  • Code 125497: Returns SBI Small Cap Fund (Labeled HDFC Top 100)
  • Code 119551: Returns Aditya Birla Banking & PSU (Labeled SBI Bluechip)
  • Code 120503: Returns Axis ELSS Tax Saver (Labeled ICICI Bluechip)
  • Code 119092: Returns HDFC Money Market (Labeled Axis Bluechip)
  • Code 120841: Returns quant Mid Cap Fund (Labeled Kotak Bluechip)

Strategic Handling

This upstream API mapping error was proactively documented for the team. The raw fetched records were saved using their official codes to guarantee consistency during subsequent join tasks.

  • Identified codes correctly matched dataset histories.
  • Detailed mapping table created in data quality report.
